Instructions for executing the program

1. Download Chrome browser and setup the path of chromedriver.exe in the file from the below link:
https://chromedriver.chromium.org/downloads

2. Make sure to update the PATH in the program to path of the cromedriver on your system

3. Install Selenium library with command:
	pip install selenium



Important Note:
The Design of dynamic website like Aliexpress keeps on updating (changing). Whenever, the design changes the web scrapping program aslo needs to be changed according to the new design.
